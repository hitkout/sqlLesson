Оператор SQL

Оператор SELECT используется для выбора данных из базы данных. Возвращенные данные сохраняются в таблице результатов, называемой набором результатов.

SELECT column1, column2, ...
FROM table_name;

Здесь column1, column2, ... - это имена полей таблицы, из которой вы хотите выбрать данные.
Если вы хотите выбрать все поля, доступные в таблице, используйте следующий синтаксис:

SELECT * FROM table_name;
